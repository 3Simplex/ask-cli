# Ask-CLI Perk Card 🤖
An agentic NixOS assistant that bridges the gap between the user and the local AI Brain.

## Features
- **Nix-Wrapped**: Carries its own Python environment and tools (requests, rich, glow).
- **Portable**: Can be slotted into any NixOS system flake.
- **Local-First**: Configured to talk to a local llama-server instance.
